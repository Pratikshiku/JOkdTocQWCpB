### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 房屋信息
5. 房屋租赁公告
6. 个人中心
7. 我的收藏
8. 后台管理

**屋主功能：**
1. 登录
2. 注册
3. 个人中心（修改密码、个人信息）
4. 房屋信息管理
5. 租用订单管理
6. 取消订单管理
7. 保洁管理
8. 保洁取消管理
9. 安保管理
10. 安保取消管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 屋主管理
5. 房屋信息管理
6. 类型管理
7. 租用订单管理
8. 取消订单管理
9. 保洁管理
10. 保洁取消管理
11. 安保管理
12. 安保取消管理
13. 我的收藏管理
14. 系统管理（轮播图管理、房屋租赁公告）
